# 🧠 Memory Card Flip Game

A visually engaging **Memory Card Flip Game** built using **HTML, CSS, and minimal JavaScript**.  
The project emphasizes **advanced CSS animations and layouts**, while JavaScript is used only for essential game logic such as card matching, shuffling, and timer control.

---

## 📌 Project Description

This game challenges players to flip cards and find matching pairs. Each card features a **3D flip animation**, **image-based faces**, and a clean **CSS Grid layout**. Cards are **randomly shuffled on every reload**, and a **timer runs during gameplay** and stops automatically once all cards are matched.

---

## 🛠️ Technologies Used

### HTML5
- Semantic structure (`header`, `main`)
- Reusable card components
- `data-value` attributes for match comparison

### CSS3
- CSS Grid for layout
- 3D transforms (`perspective`, `rotateY`, `preserve-3d`)
- Transitions and hover effects
- Keyframe animation for matched cards (pulse effect)
- Responsive design and image scaling

### JavaScript (Minimal Usage)
- Card flip handling
- Match validation logic
- Random card shuffling
- Timer start and stop control

---

## ✨ Features

- 🎴 3D card flip animation  
- 🖼️ Image-based card backs  
- 🔀 Random card positions on every restart  
- ✅ Matched cards remain flipped with pulse animation  
- 🚫 Click lock during card comparison  
- 🎨 Smooth hover and interaction effects  
- ⏱️ Automatic timer control  

---

## 🧠 How the Game Works

1. Click on a card to flip it  
2. Flip a second card to check for a match  
3. If the cards match:
   - They stay flipped
   - A pulse animation plays  
4. If the cards do not match:
   - Both cards flip back automatically  
5. The timer stops when all cards are matched
